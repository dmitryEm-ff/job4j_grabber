# job4j_grabber
# Цель проекта
Данный проект создавался для изучения парсинга сайтов с помощью библиотеки JSOUP и возможности зашедулить выполнение программы используя библиотеку Quartz.

### Используемые технологии
* JSOUP
* Quartz
* PostgreSQL, JDBC
* Slf4j логгирование
